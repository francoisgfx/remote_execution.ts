Class RemoteExecution

A remote execution session. This class can discover remote "nodes" (UE4 instances running Python), and allow you to open a command channel to a particular instance.

run_command

  • run_command(command: string, unattended?: boolean, execMode?: string, callback?: Function, raiseOnFailure?: boolean): void
    • Defined in remote_execution.ts:671

    Run a command remotely based on the current command connection.

    Parameters

    • command: string

      The Python command to run remotely.

    • unattended: boolean = true

      true to run this command in "unattended" mode (suppressing some UI).

    • execMode: string = MODE_EXEC_FILE

      The execution mode to use as a string value (must be one of MODE_EXEC_FILE, MODE_EXEC_STATEMENT, or MODE_EVAL_STATEMENT).

    • Optional callback: Function

      called function when getting response from remote node

    • raiseOnFailure: boolean = false

      true to raise a RuntimeError if the command fails on the remote target

    Returns void
